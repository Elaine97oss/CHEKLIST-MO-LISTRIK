Portal Checklist MO Listrik Reg V

📖 Panduan Penggunaan Dasar

  1. Pilih Form: Klik menu checklist di bawah sesuai jadwal.
  2. Isi Data & Bukti: Lengkapi data teks dan unggah foto evidence. (Data Anda otomatis tersimpan di HP/PC yang sedang digunakan).
  3. Validasi: Jika ada kotak berwarna MERAH, berarti kolom/foto tersebut belum diisi.
  4. Cetak Laporan: Klik tombol "🖨️ Cetak PDF".
  5. Simpan PDF: Atur *Destination/Printer* ke Save as PDF dan ukuran kertas A4.
  6. Saat print pastikan hilangkan ceklist pada header dan footer agar tidak muncul tanggal pembuatannya.

✨ Keunggulan Sistem

  • 💾 Auto-Save: Mencegah data hilang jika browser tertutup.
  • 🔄 Cross-Device: Bisa dikerjakan sebagian di HP, lalu dilanjutkan & dicetak dari Komputer.
  • 🛡️ Deteksi Foto Dobel: Mencegah kecurangan/human error dalam mengunggah foto evidence yang sama.
  • 🗜️ Kompresi Cerdas: Foto besar dari HP otomatis dikompres (~110 DPI) agar PDF tetap ringan.
  • ⚡ Offline Mode: Bekerja tanpa perlu server/internet (setelah halaman terbuka).

🔄 Cara Lanjut Sesi di Komputer Lain/Save Pekerjaan (Export / Import)

Fitur ini digunakan jika Anda ingin mengisi form dan mengunggah foto di lapangan menggunakan HP, namun ingin merapikan dan mencetak laporannya melalui Komputer/Laptop di kantor. Admin menyarankan untuk menggunakan fitur ini, menjaga jika ada sedikit revisi dari pusat.

  1. Di HP Anda, pastikan form sudah terisi sebagian, lalu tekan tombol 💾 Simpan Sesi (Export).
  2. Sebuah file dengan akhiran .json akan terunduh ke HP Anda.
  3. Kirim file .json tersebut ke Komputer/Laptop di kantor (bisa lewat WhatsApp Web, Email, atau Kabel Data).
  4. Di Komputer, buka web portal ini dan pilih menu form yang sama.
  5. Tekan tombol 📂 Lanjut Sesi (Import), lalu pilih file .json yang baru saja Anda kirim.
  6. Selesai! Semua ketikan dan foto Anda akan otomatis muncul kembali, siap untuk dilanjutkan atau dicetak!
